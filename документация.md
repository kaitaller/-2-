# 🗝️ Maze Quest — Подземный Лабиринт

Техническая документация и руководство пользователя для консольного текстового квеста на C++.

---

## 🎮 Руководство пользователя (Инструкция по работе)

### Требования к системе
- Компилятор с поддержкой стандарта **C++17** или новее (`g++`, `clang++`, MSVC).
- Эмулятор терминала с поддержкой **ANSI-цветов** и кодировки **UTF-8** (*Windows Terminal*, *PowerShell*, терминалы *Linux/macOS*).

### Сборка и запуск проекта

#### Linux / macOS:
```bash
# Клонировать репозиторий
git clone [https://github.com/](https://github.com/)<ВАШ_ЛОГИН>/maze-quest.git
cd maze-quest

# Скомпилировать исходный код
g++ -std=c++17 -o maze_quest maze_quest.cpp

# Запустить игру
./maze_quest
