# 🗝️ Maze Quest — Подземный Лабиринт

Текстовый квест на C++. Вы играете за персонажа, оказавшегося в подземном лабиринте. Ваша задача — решить головоломки и выбраться на свободу!

---

## 🗺️ Структура игры

Игра состоит из **4 локаций**:

| # | Локация | Состояние | Головоломка |
|---|---------|-----------|-------------|
| 1 | Тёмный вход | Открыта (старт) | — |
| 2 | Зал Загадок | Закрыта | Загадка (слово-ответ) |
| 3 | Комната Счисления | Закрыта | Математика (уравнение) |
| 4 | Врата Свободы | Закрыта | Последовательность Фибоначчи |

Каждая локация отображается в терминале в виде ASCII-карты (вид сверху).  
Условные обозначения на карте:
- `@` — Игрок
- `C` — Сундук (Chest)
- `N` — NPC
- `D` — Дверь (Door)
- `#` — Стена
- `.` — Пол

---

## ⚙️ Требования

- Компилятор C++17 или новее (`g++`, `clang++`, MSVC)
- Терминал с поддержкой ANSI-цветов (Linux, macOS, Windows Terminal / PowerShell)

---

## 🚀 Сборка и запуск

### Linux / macOS

```bash
# Клонировать репозиторий
git clone https://github.com/<ВАШ_ЛОГИН>/maze-quest.git
cd maze-quest

# Скомпилировать
g++ -std=c++17 -o maze_quest maze_quest.cpp

# Запустить
./maze_quest
```

### Windows (MinGW / MSYS2)

```bash
g++ -std=c++17 -o maze_quest.exe maze_quest.cpp
maze_quest.exe
```

### Windows (MSVC / Developer Command Prompt)

```bash
cl /EHsc /std:c++17 maze_quest.cpp /Fe:maze_quest.exe
maze_quest.exe
```

---

## 🎮 Управление

| Ввод | Действие |
|------|----------|
| `<номер локации>` | Перейти в указанную локацию |
| `q` | Выйти из игры |

При попытке войти в **закрытую** локацию откроется головоломка.  
На решение даётся **3 попытки**.  
После верного ответа локация открывается навсегда.
